# 超级小说家一致性管理器 v1.0

> **核心理念**：长篇小说最大的敌人不是写得差，而是写着写着崩了。

全生命周期的一致性守卫，为百万字长篇装上**免疫系统**。

## 四大守卫维度

| 守卫 | 职责 | 检查周期 |
|------|------|----------|
| 📋 伏笔守卫 | 埋→记→触→收 全生命周期管理 | 每章 |
| 🎭 人物守卫 | OOC检测 + 弧光追踪 + 关系网络 | 每章 |
| 🌍 世界守卫 | 规则一致 + 时间线 + 空间验证 | 每章 |
| 📜 剧情守卫 | 因果链追踪 + 断点检测 + 多线协调 | 每5章 |

## 核心能力

- **伏笔追踪表**：编号/类型/关键程度/预计回收/状态 六维管理
- **OOC检测引擎**：7维度自动检测（性格/能力/语言/动机/关系/知识/情感）
- **健康度监控**：≥80%健康 / 60-79%警告 / <60%危险
- **五章深度审查**：伏笔→人物→世界→剧情 四步全面审查
- **结局一致性终极检查**：伏笔/人物/世界/剧情/体验 五维收束

## 与其它 Skill 协作
```
novelist-workflow-engine → STEP-004 调用一致性检查 → novelist-consistency-manager
                                    ↓
                              STEP-006 写入伏笔/人物状态
```
